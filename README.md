# Tip Calculator App

A responsive tip calculator built with React and Vite.

## How to Run

Install dependencies:

npm install

Run development server:

npm run dev

Open in browser:

http://localhost:5173

## Tech Stack

- React
- Vite
- CSS

## Features

- Calculate total tip
- Calculate amount per person
- Preset tip percentage buttons
- Custom tip input
- Responsive design for mobile and desktop
